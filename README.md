### Bluecore Development with Docker

To start Marva in a development environment using Docker, you can use the provided `docker-compose-dev.yaml` file. This file sets up the necessary services for development, including the Marva application and its dependencies.
```bash
  docker compose -f docker-compose-dev.yaml up
```
